steps to push in git repo :

step1 : git init
step2 : check the username (git config user.name), if you wanna change and push it to your another github account , then -> git config user.name "username" (avi0080 & rajeevJha-0700)
step3 : similarly check for the user email (git config user.email) and change it accordingly if you want to (avi5600sharma@gmail.com & rajeevranjanjha0001@gmail.com)
step4 : git add .
step5 : git commit -m "message"
step6 : git remote add origin git@github.com:(bla bla bla, whatever generated by github)
step7 : git remote set-url origin git@github-avi:avi0080/repo_name.git 
                   OR
        git remote set-url origin git@github-rajeev:rajeevJha-0700/repo_name.git

step8 : git push -u origin main

once you finish it for the first time , then after every modification in file can be handled by three simple steps 
 git add "file_path" 
 git commit -m "message"
 git push
